# UVM Practice — Variant 23

Подготовка проекта для выполнения учебной практики.

## Интерпретатор (Этап 3)

Интерпретатор выполняет машинный код УВМ с раздельной памятью команд и данных.
Модель вычислений — стековая.

### Поддерживаемые команды (этап 3)
- LOAD_CONST — загрузка константы в стек
- READ_MEM — чтение значения из памяти по адресу со стека
- WRITE_MEM — запись значения в память

## Операция <= (Этап 4)

Команда `LE` выполняет сравнение `left <= right` и записывает результат (1 или 0)
в память по адресу `addr`, полученному со стека.

Пример: `examples/stage4_le_demo/le_demo.yaml`

## Этап 5: тестовая задача варианта 23

Выполняется поэлементное сравнение `<=` вектора длины 7 (адреса 100..106) с числом 233.
Результат (1 или 0) записывается в исходный вектор.

Примеры программ:
- `examples/stage5_task/case1_all_less.yaml` — ожидаемо все элементы становятся 1
- `examples/stage5_task/case2_all_greater.yaml` — ожидаемо все элементы становятся 0
- `examples/stage5_task/case3_mixed.yaml` — ожидаемо: 1,1,0,1,0,1,0

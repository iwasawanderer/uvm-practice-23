# UVM Practice — Variant 23

Подготовка проекта для выполнения учебной практики.

## Интерпретатор (Этап 3)

Интерпретатор выполняет машинный код УВМ с раздельной памятью команд и данных.
Модель вычислений — стековая.

### Поддерживаемые команды (этап 3)
- LOAD_CONST — загрузка константы в стек
- READ_MEM — чтение значения из памяти по адресу со стека
- WRITE_MEM — запись значения в память
